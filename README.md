# Stim Tycoon

Stim Tycoon is a Unity game project scaffold.

## Getting Started

1. Install Unity Hub.
2. Open this repository as a Unity project.
3. Use the Unity version listed in `ProjectSettings/ProjectVersion.txt` when possible.

## Project Layout

- `Assets/Scenes` — Unity scenes.
- `Assets/Scripts` — C# gameplay scripts.
- `Assets/Art` — sprites, models, materials, and other art assets.
- `Assets/Audio` — music and sound effects.
- `Assets/Prefabs` — reusable Unity prefabs.
- `Packages` — Unity Package Manager manifest and lock file.
- `ProjectSettings` — Unity project settings that should be committed.

## Version Control Notes

Unity-generated folders such as `Library`, `Temp`, `Obj`, `Build`, `Builds`, `Logs`, and `UserSettings` are ignored. Commit source assets, package files, project settings, and `.meta` files.
